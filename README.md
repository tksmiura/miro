# miro

View markdown on console using ANSI escape code

## install

Copy **miro** to directory thought path.

If you need code syntax high light, Install source-highlight.
like this.

```
# apt install source-highlight
```

## usage

```
$ miro <markdown file>
```

using with less

```
$ miro -l <markdown file> | less -R
```
